##  Project Overview
Shopping in modern retail environments relies heavily on visual interfaces, creating accessibility challenges for blind and elderly users. This project presents a Voice-Assisted Virtual Cart that enables independent shopping through voice interaction, computer vision, and AI-powered assistance.

The system integrates speech recognition, real-time object detection, automated billing, and text-to-speech feedback to provide an inclusive, hands-free shopping experience.

##  Objectives
- Enable voice-based interaction for blind and elderly users
- Provide real-time product detection using computer vision
- Offer auditory feedback for all actions
- Automate cart management and billing
- Build a low-cost, browser-based solution

---

##  Features
- Voice command processing
- Real-time object detection (COCO-SSD)
- Virtual cart management
- Automated billing with GST
- Text-to-speech feedback
- AI interaction using Google Gemini API
